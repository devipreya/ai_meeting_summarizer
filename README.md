# AI Meeting Summarizer

An AI-powered meeting assistant that converts meeting audio into structured summaries, extracts action items, and generates downloadable PDF reports using local AI models.

## Features

- Upload meeting audio files
- Audio-to-text transcription using Whisper
- AI-generated meeting summaries
- Automatic action item extraction
- PDF report generation
- MongoDB storage for meeting history
- Modern React frontend dashboard
- Fully local AI inference using Ollama + Phi-3

## Tech Stack

### Frontend
- React
- React Router
- Tailwind CSS

### Backend
- Flask
- Python
- FFmpeg
- Whisper

### AI Models
- Ollama
- Phi-3

### Database
- MongoDB

### PDF Generation
- ReportLab
